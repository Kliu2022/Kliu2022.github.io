Activity 2: JSON Data - HIV Infection Rate By Country

Stage Link Short Description
Stage 1
In this stage I checked to ensure that the site was properly loading the JQUERY resources. The JQUERY library will be helpful for many Javascript tasks, in this case reading from a JSON file. This will probably be used for my final website as well.
Stage 2
In this stage I accessed and stored the JSON data using a static URL with the JSON file. As I will be reading from raw data in JSON form for my website it will be very important to learn the structure of a JSON file.
Stage 3
In this stage, I used the console to explore the structure of the JSON data and was able to read specific information from the JSON file, represented as a large array. Once again, it is very important to understand the structure of a JSON file.
Stage 4
This stage involved using Javascript functions to manipulate the string of data into an integer. This involved using functions and methods such as .substring(), .indexOf(), and .replace(). I then displayed the first few values from the array graphically on the site with bars representing the number of people living with HIV. Some math was necessary to scale the lengths of the bars to the size of the window. In general, this stage involved manipulating and graphically displaying data from the JSON file. These methods and techniques will definitely benefit my development of the final website where data will be graphcially displayed.
Stage 5
In this stage, I learned how to use for loops in Javascript to display all the integer values in the array graphically. The same functions and math in Stage 4 were applied to each element of the array of data using a for loop and displayed to the screen in a different for loop. The stage also involved using a simple algorithm to find the largest value of the array of integers. Throughout the development of my final site, many repetitive tasks will be necessary and as such using loops is a crucial skill to have.
Stage 6
In this stage, I added a fixed header on the page. Additionally, I used the modulus function to alternate the colours of the bars between four colours, rather than making them all the same colour. My website will incorporate both of these skills.
Stage 7
Finally, in Stage 7, I used a user input field to display only the countries with a number of people living with HIV less than the number the user entered. This required a new condition in an if statement as well as a new function to generate the display. Allowing a user to interact with the website is the next step with developing my website. I will aim to allow the user to interact with the website in some way.
